## Setup Instructions

### Backend
cd server
npm install

Create a .env file inside the server folder:

env:
PORT=5000
MONGO_URI=yourconnectionstring


Run backend:


node server.js

### Frontend

cd client/PocketGuard
npm install
npm run dev

Open:
http://localhost:5173

## Features

Add transactions
Edit transactions
Delete transactions
Income/expense tracking
